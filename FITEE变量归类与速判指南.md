# FITEE-2025-0154 变量归类 + LaTeX 字体速判法

---

## 📖 一、核心判断原则（速记版）

把每个字母/符号过一遍"三问"：

```
问题1：它是几个字母？ → 多字母缩略词？→ \mathrm{} 正体
                       → 单个字母？   → 进入问题2

问题2：它是变量还是常量/函数/运算符？
                      → 运算符(=, +, ∑, ∂)、已知函数(sin, σ, tanh)、数字 → 正体
                      → 变量 → 进入问题3

问题3：它是向量/矩阵/张量吗？（单字母能"加粗"）
                      → 是 → \bm{} 粗斜体
                      → 否 → 普通斜体（不写任何命令）
```

**三个最重要的一票否决：**
- 看到 **multi-letter 缩略词**（MP-DMC, RL, MAE, Gbps）→ 立刻 `\mathrm{}` 正体
- 看到 **e, π, d（微分）, j（虚部）** → 立刻正体 `\mathrm{e}`, `\mathrm{d}` 等
- 下标是**单词缩写**（pre, max, opt）不是变量 → 立刻 `\mathrm{pre}`

## 三、速判流程图

```
看到符号 → 
├─ 单位（Gbps/GB/s/ms/vCPUs） → \mathrm{} 正体
├─ 多字母缩略词（MAE/RMSE/Loss/DMA-LSTM/HPA/CLB） → \mathrm{} 正体 或 \text{}
├─ 已知函数（sin/cos/tanh/exp/log/σ/max/min） → 正体（\sin、\sigma、\text{max}）
├─ 自然常量（e/π/i/j/d微分） → 正体（\mathrm{e}、\mathrm{d}）
├─ 数字、括号、运算符（=、+、,、(、∑） → 正体（自带正体）
├─ 单字母 →
│   ├─ 向量/矩阵/张量 → \bm{} 粗斜体
│   ├─ 空心（期望/集合） → \mathbb{}
│   ├─ 花体 → \mathcal{}
│   └─ 普通标量 → 斜体（不加命令）
├─ 下标 →
│   ├─ 单字母变量下标（_i, _j, _t, _n） → 斜体
│   └─ 多字母/缩写下标（_pre, _thr, _opt, _max） → \mathrm{} 正体
├─ 上标 →
│   ├─ 单字母变量（^T 转置除外） → 斜体
│   ├─ 单词缩写（^pre, ^opt） → \mathrm{} 正体
│   └─ 转置 ^T → 正体（但通常用 ^\top 或 ^{\mathrm{T}}）
└─ 希腊字母 →
    ├─ 普通用法 → 斜体（\beta, \alpha, \xi, \sigma 做变量时）
    └─ 做单位/算子 → 正体（\mathrm{} 包裹，极少情况）
```

## 四、本文中你有能力立即自查的常见陷阱

| 陷阱 | 错误示例 | 正确写法 | 判断口诀 |
|------|---------|---------|---------|
| 多字母缩略词未正体 | `MAE` | `\mathrm{MAE}` | "超过一个字母的缩写→正体" |
| 缩写下标未正体 | `_c` (c=consumed) | `_{\mathrm{c}}` | "下标代表单词→正体" |
| 单字母向量未加粗 | `W_f` | `\bm{W}_f` | "W是权重矩阵→能加粗就加粗" |
| 隐状态h未加粗 | `h_t` | `\bm{h}_t` | "h是多个数的向量→加粗" |
| 数学期望未用空心 | `E[x]` | `\mathbb{E}[x]` | "期望E→空心正体" |
| 变量和下标分不清 | `L_\beta` vs `\beta` | β是希腊变量→斜体 | "希腊字母做变量→斜体" |
| 函数和变量混淆 | `σ(x)` / 做变量用它 | 做sigmoid→正体；做标准差→斜体 | "同一字母不同语境不同字体" |
